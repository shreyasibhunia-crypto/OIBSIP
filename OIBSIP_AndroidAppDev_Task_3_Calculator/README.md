## 📱 Features

- ✔️ **Addition:** Perform standard addition operations seamlessly.
- ✔️ **Subtraction:** Execute accurate subtraction calculations.
- ✔️ **Multiplication:** Quickly multiply numbers.
- ✔️ **Division:** Divide with precision and robust error handling (e.g., divide by zero).
- ✔️ **Decimal Calculations:** Support for floating-point arithmetic for precise results.
- ✔️ **Clear Button:** Instantly clear the screen to start a new calculation.
- ✔️ **User-Friendly UI:** A modern, clean, and intuitive interface designed for ease of use.

## 📂 Project Structure

- **`app/src/main/java/`**: Contains the core Java logic managing user input, state, and mathematical operations.
- **`app/src/main/res/layout/`**: Contains the XML files defining the visual grid and layout of the calculator buttons and display.
- **`app/src/main/res/drawable/`**: Contains custom button shapes, colors, and interactive ripple effects.
- **`app/src/main/res/values/`**: Contains dimensions, colors, strings, and overall application themes.
- **`build.gradle`**: Manages build configurations, SDK versions, and project dependencies.

## 🚀 Installation Guide

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   ```
2. **Open in Android Studio:**
   Launch Android Studio, go to `File > Open`, and select the `CALCULATOR` directory.
3. **Sync Gradle:**
   Wait for Android Studio to sync and configure the build dependencies. Click `Sync Project with Gradle Files` if prompted.
4. **Run the application:**
   Connect your Android smartphone or use an emulator, and press the `Run` ▶️ button to deploy.

## 📖 Learning Outcomes

During the creation of this Calculator app, I developed practical skills in:
- Constructing complex UI grids using **ConstraintLayout** and `LinearLayout` in XML.
- Capturing and processing user inputs from multiple buttons via Java event listeners.
- Evaluating mathematical expressions and managing string concatenation and conversion.
- Understanding how to gracefully handle edge cases such as trailing decimals or division by zero.
- Applying basic software architecture principles to separate UI design from calculation logic.
