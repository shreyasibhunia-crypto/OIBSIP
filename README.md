## 📱 Project Overview

### 📏 Unit Converter
A highly efficient unit conversion application designed with a simple and intuitive UI. It allows users to quickly convert values across multiple metric and imperial categories with fast performance and accurate results.

### 🧮 Calculator
A functional calculator application that supports standard arithmetic operations. Built with a clean interface and responsive design, ensuring smooth calculations and an excellent user experience.

### ⏱️ Stopwatch
A modern stopwatch application featuring start, pause, resume, reset, and lap functionalities. The intuitive layout makes it perfectly suited for tracking time efficiently in everyday scenarios.

## 🚀 Getting Started

Follow these steps to set up the projects on your local machine:

**1. Clone the repository:**
```bash
git clone https://github.com/shreyasibhunia-crypto/OIBSIP.git
```

**2. Open in Android Studio:**
- Launch Android Studio.
- Click on `File` > `Open`.
- Navigate to the cloned repository and select the specific project folder (e.g., `OIBSIP_AndroidAppDev_Task_1_UnitConverter`).

**3. Sync Gradle:**
- Wait for Android Studio to index the files and configure the project.
- Click on the `Sync Project with Gradle Files` icon if prompted.

**4. Run the app:**
- Connect your Android device or start an emulator.
- Click the `Run` ▶️ button in Android Studio to build and install the app.
