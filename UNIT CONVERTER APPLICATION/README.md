## 📱 Features

- ✔️ **Convert Length:** Meters, Kilometers, Miles, etc.
- ✔️ **Convert Weight:** Grams, Kilograms, Pounds, etc.
- ✔️ **Convert Temperature:** Celsius, Fahrenheit, Kelvin.
- ✔️ **Convert Time:** Seconds, Minutes, Hours, Days.
- ✔️ **Convert Speed:** km/h, mph, m/s.
- ✔️ **User-friendly interface:** Clean and intuitive UI.
- ✔️ **Instant conversion:** Real-time result calculation.
- ✔️ **Responsive design:** Adapts to various screen sizes perfectly.

## 📂 Project Structure

- **`app/src/main/java/`**: Contains the core Java source code, including logic for UI interaction and mathematical conversions.
- **`app/src/main/res/layout/`**: Contains XML layout files defining the user interface structure.
- **`app/src/main/res/drawable/`**: Contains visual assets, icons, and background resources.
- **`app/src/main/res/values/`**: Contains strings, colors, and styles for the app.
- **`build.gradle`**: Contains app-level and project-level build configurations and dependencies.

## 🚀 How to Run

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   ```
2. **Open in Android Studio:**
   Launch Android Studio, select `File > Open`, and choose the `UNIT CONVERTER APPLICATION` folder.
3. **Sync Gradle:**
   Click the `Sync Project with Gradle Files` button to download dependencies.
4. **Run the application:**
   Connect a device or launch an emulator, then click the `Run` ▶️ button.
